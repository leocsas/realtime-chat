# Realtime Chat

Chat em tempo real utilizando WebSocket, desenvolvido com HTML, CSS, JavaScript e Node.js.

## Objetivo do projeto
Praticar a comunicação em tempo real utilizando WebSocket, entendendo o fluxo de mensagens entre frontend e backend, além do uso de eventos e manipulação do DOM com JavaScript.

## Tecnologias
- HTML
- CSS
- JavaScript
- Node.js
- WebSocket

## Como executar
```bash
npm install
npm run dev
